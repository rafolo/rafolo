# Sample app

Sample geolocalisation application

## Bower install
Run
    ~$ npm install
    ~$ npm install -g bower
    ~$ bower install

## Karma install
    ~$ npm install -g karma
    ~$ npm install -g karma-cli

## Protractor install
    ~$ npm install -g protractor

## Tests
Run
    ~$ karma start
    ~$ protractor

Troubleshooting:

Protractor not starting on Windows:

    ~$ webdriver-manager start --standalone
